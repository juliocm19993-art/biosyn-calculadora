# Calculadora BioSYN

## Rodar localmente
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy no Vercel
- Suba este projeto no GitHub
- Importe o repositório na Vercel
- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
